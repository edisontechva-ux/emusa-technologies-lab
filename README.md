# EMusa Technologies Lab

Hands-on IT administration lab built to practice real-world **Microsoft 365 Administration** 
and **Identity & Access Management (IAM)** scenarios that SMBs and mid-size orgs actually 
run into — from tenant setup to access governance.

## About this lab

I built this using a live Microsoft 365 tenant (not a trial sandbox) to simulate the kind 
of work an M365 Administrator or IAM System Admin handles day to day: identity security, 
device compliance, mail flow governance, and access control across a growing organization.

Each project below includes the business scenario, what I configured, why I made those 
decisions, and proof of the working result (scripts, policy exports, or screenshots).

## Skills demonstrated

**Microsoft 365 Administration:** Exchange Online, Microsoft Entra ID, Intune endpoint 
management, Security & Compliance Center, Microsoft Graph API/PowerShell automation

**Identity & Access Management:** Conditional Access policies, MFA enforcement, 
directory services (Entra ID), access reviews, SSO concepts, role-based access control (RBAC)
